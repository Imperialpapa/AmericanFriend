# EngFriend — AI Language Learning App

AI tutor "Alex"와 자연스러운 대화를 통해 외국어를 배��는 앱입니다.

## Key Features
- 4개 AI 프로바이더 (Gemini, Groq, Claude, OpenAI) + 모델 선택
- 9개 언어 지원 (영어, 중국어, 한국어, 일본어, 스페인어, 프랑스어, 이탈리아어)
- 10단계 레벨 시스템 (자동 평가)
- 음성 대화 (STT + TTS)
- 주제 집중 모드 + 역할극 미션
- 단어장 (Spaced Repetition)
- 일일 연속 학습 (Streak) + 주간 리포트

## Quick Start
```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```
