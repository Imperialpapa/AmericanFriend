## 1. 수익 모델

### 프리미엄 (Freemium)
| 구분 | 무료 | 프리미엄 (월 ₩4,900~9,900) |
|------|------|---------------------------|
| 일일 대화 | 10회 | 무제한 |
| AI 모델 | Gemini Flash | Claude/GPT-4o 선택 가능 |
| 발음 평가 | 3회/일 | 무제한 |
| 레벨 테스트 | 기본 | 4영역 상세 리포트 |
| 대화 주제 | 일상 대화만 | 비즈니스, TOEIC, 면접 등 |
| 광고 | 배너 | 없음 |
| 대화 히스토리 | 최근 7일 | 무제한 보관 |

### 대안 수익 모델
- **API 키 자체 사용 모드**: 사용자가 자신의 API 키를 입력하면 무료 (현재 방식)
- **구독 모드**: 앱이 API 비용을 부담, 사용자는 월정액만 지불
- **하이브리드**: 무료는 자체 키 입력, 프리미엄은 키 없이 바로 사용

## 2. API 비용 관리 (서버 아키텍처)

### 현재: 클라이언트 직접 호출
```
[앱] → [Gemini API] (사용자 API 키)
```
- 장점: 서버 비용 zero
- 단점: API 키 노출 위험, 사용량 제어 불가

### 상용화: 백엔드 서버 경유
```
[앱] → [백엔드 서버] → [AI API]
```
- **백엔드 역할**: 인증, API 키 관리, 사용량 추적, 과금
- **추천 스택**: Firebase (Auth + Firestore + Cloud Functions) 또는 Supabase
- **비용 최적화**:
  - 응답 캐싱 (같은 레벨의 제안 문장 등)
  - Gemini Flash 기본 + 프리미엄만 고급 모델
  - 프롬프트 토큰 최적화 (대화 히스토리 요약)

### 예상 API 비용 (Gemini Flash 기준)
| 사용자 수 | 일일 대화 | 월간 API 비용 (추정) |
|-----------|-----------|---------------------|
| 100명 | 10회/인 | ~$3~5 |
| 1,000명 | 10회/인 | ~$30~50 |
| 10,000명 | 10회/인 | ~$300~500 |

Gemini Flash는 매우 저렴하므로 초기 스케일에서 부담 적음.

## 8. 경쟁 분석

| 앱 | 특징 | 가격 | 우리의 차별점 |
|-----|------|------|-------------|
| Speak | AI 튜터, 롤플레이 | ₩14,900/월 | 자연스러운 친구 대화, 저렴 |
| ELSA Speak | 발음 평가 특화 | ₩11,000/월 | 대화 + 발음 통합 |
| ChatGPT | 범용 AI | $20/월 | 영어 학습 특화 UX, 한국인 맞춤 |
| Cake | 영상 + 대화 | ₩7,900/월 | 실시간 AI 대화, 레벨 자동 조절 |

### 우리의 핵심 경쟁력
1. **자연스러운 친구 느낌** — 튜터가 아닌 미국인 친구와의 대화
2. **한국인 특화** — 한글 번역, 한국 문화 이해, L/R 등 발음 피드백
3. **저렴한 진입장벽** — 무료 API 키로 시작 가능
4. **오픈소스 기반** — 투명성, 커뮤니티 기여 가능
