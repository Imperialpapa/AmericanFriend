# Redesign Brief — Korean Friend

> Claude Design에 전달할 리디자인 목표/톤 브리프
> 작성일: 2026-04-23
> ⚠️ **사용자 확인 필요** — 아래 항목들은 분석 기반 초안. 컨펌 후 A4 패키지에 반영.

## 1. 제품 본질

- **앱 한 줄 설명**: AI 캐릭터 "Alex"와 자연스러운 대화를 통해 외국어를 배우는 모바일 앱
- **타겟 사용자**: 한국에서 영어를 배우려는 20~40대 성인 학습자 (직장인/대학생 중심)
- **핵심 가치**: "선생님보다 친구처럼" — 부담 없이 매일 짧게 대화 연습
- **1차 사용 시나리오**: 출퇴근 지하철, 자기 전 침대, 점심시간 잠깐 등 모바일 자투리 시간

## 2. 리디자인 목표

| 우선순위 | 목표 | 측정 지표 (가설) |
|---|---|---|
| ★★★ | "친구 같은 따뜻한 AI"라는 브랜드 인상 강화 | 첫 화면 인상 호감도, 리텐션 |
| ★★★ | 학습이 "재미있어 보이게" — 게이미피케이션 시각화 | streak/level/mission UI 강화 |
| ★★ | 정보 밀도 정리 — Settings/AppBar 과밀 해소 | 설정 도달 시간 단축 |
| ★★ | 대화 화면을 모바일 채팅 앱처럼 익숙하면서도 차별화 | 메시지 가독성 |
| ★ | 다크모드 일관성 + 컬러 팔레트 확장 | 다크모드 사용률 |

## 3. 디자인 톤 키워드

**선택**: `warm` + `playful` + `trendy modern` + `confident`

- **Warm**: 학습 부담을 줄이는 따뜻한 색감 (cool blue 단일 → warm accent 추가)
- **Playful**: 캐릭터 일러스트, 마이크로 인터랙션, 보상감 (단, 유아용 느낌은 X)
- **Trendy modern**: 2026 트렌드 — 두꺼운 라운드, soft gradient, neumorphism은 X, 명확한 그림자
- **Confident**: 진지한 학습 가치도 전달 (장난감 같은 느낌은 X)

## 4. 레퍼런스 (제안)

> ⚠️ 사용자 추가/제거 환영

- **Duolingo** — 캐릭터 중심, 보상감, 일러스트, 게이미피케이션 (단, 너무 만화풍은 피함)
- **Cake** — 한국 영어학습 앱, 모바일 채팅 UX 참고
- **ChatGPT mobile** — 깔끔한 채팅 UI, 메시지 버블 처리
- **Linear** — 깔끔한 미니멀, 다크모드 처리, 디테일한 타이포
- **Headspace** — warm + playful 톤, 일러스트 활용
- **Anthropic Claude** — 따뜻한 미니멀, 베이지/오렌지 톤

## 5. 절대 피해야 할 것

- ❌ "Generic AI 앱" 느낌 (단조로운 그라디언트, 보라색 일변도)
- ❌ 차가운 테크 톤 (블루+회색 일변도, B2B 같은 진지함)
- ❌ 유아용 느낌 (과한 캐릭터, 만화풍 폰트)
- ❌ 광고에 묻히는 디자인 (배너 광고와 시각적 충돌 최소화)
- ❌ 다크모드만 신경 쓴 디자인 (라이트모드도 1급)

## 6. 컬러 팔레트 방향성 (제안)

**기존 단일 시드(`#4A90D9` 차분한 블루) → 다중 토큰 시스템으로 확장**

제안 방향 (Claude Design에 옵션으로 제시):
- **Option A — Warm Coral**: 메인 코랄/오렌지, 보조 크림, 액센트 인디고
- **Option B — Modern Sage**: 메인 세이지 그린, 보조 베이지, 액센트 머스타드
- **Option C — Trendy Lavender**: 메인 부드러운 라벤더, 보조 피치, 액센트 다크 네이비

> Claude Design에서 3개 모두 받아보고 사용자 선택

## 7. 캐릭터 "Alex" 일러스트 방향

**현재**: CircleAvatar 'A' 텍스트만 — 브랜드 부재

**제안**:
- 성별 중립적 캐릭터 (현재 Lottie는 "talking girl") — 또는 male/female 둘 다
- 친근하고 trendy한 일러스트 스타일 (Notion 스타일 또는 Memoji 풍)
- AppBar용 작은 버전 + Welcome용 큰 버전 + 감정 표현 변형 (happy/thinking/celebrate)

> Claude Design에 일러스트 생성 가능 여부 따라 결정. 안 되면 향후 별도 일러스트레이터 작업

## 8. 우선 리디자인 화면 (3개 선정)

A4 패키지에 포함할 화면:

1. **ChatScreen** ⭐ — 메인 화면, 사용 빈도 최고
2. **OnboardingScreen** ⭐ — 첫인상 결정
3. **SettingsScreen** ⭐ — 정보 밀도 정리 효과 큼

선정 근거:
- Chat은 데일리 사용, 디자인 영향이 가장 큼
- Onboarding은 첫인상이 리텐션에 직결
- Settings는 정보 정리만으로도 큰 개선 (Weekly Report/Vocabulary 분리 검토)

## 9. 제약사항

- **광고 배너 유지** (모든 주요 화면 하단) — 광고와 시각적 충돌 없게
- **Material 3 위에서 동작** — 완전 커스텀 위젯은 비용 큼, ThemeData 확장 우선
- **다크모드 필수**
- **다국어 대응** — 텍스트 길이 가변 (영어/한국어 차이)
- **최소 SDK**: 기존 환경 유지 (Flutter SDK ^3.11.3)
