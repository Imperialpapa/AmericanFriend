# American Friend - 추가 기능 할일 목록

## 우선순위 높음
- [ ] **발음 평가** — 사용자 음성 녹음 → 발음 정확도 피드백
- [ ] **레벨 테스트 확장** — 주기적 실력 측정 → 레벨 자동 조정

## 우선순위 중간
- [ ] **단어장** — 대화 중 새 단어 자동 저장, 복습 기능
- [ ] **문법 교정 리포트** — 대화 후 틀린 문법 정리 & 피드백
- [ ] **대화 주제 선택** — 카페 주문, 여행, 면접 등 상황별 시나리오
- [ ] **대화 히스토리** — 과거 대화 목록 조회 & 이어서 대화하기

## 우선순위 낮음
- [ ] **일일 학습 목표** — 하루 대화 횟수/시간 목표 & 달성 트래커
- [ ] **퀴즈 모드** — 대화에서 배운 표현으로 빈칸 채우기 퀴즈
- [ ] **표현 노트** — 관용표현/슬랭 북마크, 예문과 함께 저장
- [ ] **레벨 테스트** — 주기적 실력 측정 후 레벨 자동 조정
